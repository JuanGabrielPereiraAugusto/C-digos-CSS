Responsividade no CSS

Um resumo em formato de site para revisar os fundamentos e aplicar em projetos reais: mobile first, media queries, Flexbox, Grid, imagens fluidas e tipografia adaptativa.

O que e responsividade?

Capacidade do layout de se adaptar a varios tamanhos de tela mantendo leitura e usabilidade.

Meta viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

Unidades ideais

  • %
  • vw e vh
  • rem
  • fr (Grid)

1) Mobile First

Comece para celular e depois expanda com breakpoints de tela maior usando min-width. 

.card {
  padding: 1rem;
  font-size: 1rem;
}

@media (min-width: 768px) {
  .card {
    padding: 1.5rem;
    font-size: 1.125rem;
  }
}

2) Breakpoints e Media Queries

Valores comuns para guiar testes (nao sao obrigatorios): 480px, 768px, 1024px e 1280px. 

.container {
  width: min(100% - 2rem, 1100px);
  margin-inline: auto;
}

@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

3) Flexbox e Grid no dia a dia

Flexbox e melhor para alinhamento em uma dimensao. Grid e melhor para estrutura completa em linhas e colunas. 

.menu {
  display: flex;
  flex-wrap: wrap;
  gap: 0.75rem;
}

.cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1rem;
}

4) Imagens e Tipografia fluida

img {
  max-width: 100%;
  height: auto;
  display: block;
}

h1 {
  font-size: clamp(1.5rem, 2.5vw, 2.5rem);
}
Use clamp() para titulos Use max-width: 100% em imagens Evite travar tudo em px

5) Checklist rapido

  • A pagina funciona em 320px?
  • Texto legivel sem zoom?
  • Botoes com area boa para toque?
  • Imagens sem quebrar layout?
  • Layout realmente mobile first?

6) Exercicio pratico

  1. Monte um layout com cabecalho, cards e rodape.
  2. Mobile: 1 coluna.
  3. Tablet (>= 768px): 2 colunas.
  4. Desktop (>= 1024px): 3 colunas.
  5. Use clamp() no titulo principal.
