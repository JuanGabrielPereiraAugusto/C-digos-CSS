Guia de Estilizacao Basica com CSS

1) Alterando o Fundo dos Elementos

Aqui voce define o fundo com cor solida ou imagem. Use background-color para cor e background-image para imagem ou gradiente.

background-color
background-image

2) Redimensionando as Imagens de Fundo

cover preenche toda a caixa (pode cortar), contain mostra tudo (pode sobrar espaco) e tamanho fixo usa valores exatos em pixels.

background-size: cover
background-size: contain
background-size: 80px 80px

3) Repeticao das Imagens de Fundo

Controla quantas vezes a imagem repete. Use repeat para repetir em ambos os eixos, no-repeat para nao repetir e repeat-x para repetir apenas na horizontal.

background-repeat: repeat
background-repeat: no-repeat
background-repeat: repeat-x

4) Posicionamento das Imagens de Fundo

Define onde a imagem de fundo aparece dentro da caixa. Pode ser por palavras-chave (left, center, right) ou por valores numericos.

background-position: left top
background-position: center
background-position: right bottom

5) Propriedade Background Attachment

Define se o fundo rola com o elemento (scroll), com o conteudo interno (local) ou fica preso na tela (fixed).

attachment: scroll

role para testar

attachment: local

role para testar

attachment: fixed

role para testar

6) Propriedade Background Origin

Define de onde comeca o calculo de posicionamento do fundo: border-box, padding-box ou content-box.

origin: border-box
origin: padding-box
origin: content-box

7) Propriedade Background Clip

Define ate onde o fundo e pintado. Com isso voce escolhe se a pintura pega borda, apenas padding ou apenas a area de conteudo.

clip: border-box
clip: padding-box
clip: content-box

8) Mesclagem

background-blend-mode mistura camadas de fundo (gradiente + imagem) para criar efeitos visuais sem editar a foto.

background-blend-mode: multiply
background-blend-mode: screen

9) Propriedade Background (Shorthand)

A forma abreviada junta varias propriedades em uma unica linha: cor, imagem, repeticao, posicao e tamanho.

background: cor + imagem + repeat + position/size

10) Cores e Texto

Este bloco mostra as propriedades mais usadas para tipografia: cor, tamanho, peso, alinhamento, decoracao, espacamento entre linhas e familia da fonte.

11) Box Model

Todo elemento tem conteudo, padding, borda e margem. content-box e border-box mudam como largura e altura finais sao calculadas.

content-box
border-box

12) Display e Posicionamento

display define o comportamento do elemento na linha. position controla deslocamento relativo, absoluto e a ordem de camadas com z-index.

inline inline block inline-block
position: relative
position: absolute
z-index 1
z-index 2

13) Unidades

Use px para tamanho fixo, % relativo ao elemento pai, rem relativo a fonte raiz e vw relativo a largura da viewport.

140px
70%
12rem
25vw

14) Layout Moderno (Flexbox e Grid)

Flexbox e ideal para alinhamento em uma dimensao (linha/coluna). Grid e ideal para montar estruturas em duas dimensoes (linhas e colunas).

15) Estados e Interacoes

Pseudo-classes mudam o estilo conforme a acao do usuario: :hover, :focus e :active. Isso melhora feedback visual e usabilidade.

16) Efeitos Simples

Efeitos comuns para acabamento visual: cantos arredondados, sombra, transparencia e animacao suave com transition.

box-shadow
opacity
