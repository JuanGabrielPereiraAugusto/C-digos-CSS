CSS Grid: o que voce precisa saber

Grid e um sistema de layout bidimensional (linhas e colunas). Use quando precisar controlar estrutura completa da pagina, cards, dashboards e areas como cabecalho/conteudo/barra lateral.

display: grid grid-template-columns grid-template-areas gap auto-fit + minmax

1) Conceito base: container e itens

O elemento pai vira o grid container com display: grid. Filhos diretos viram itens da grade.

.container {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
  gap: 0.55rem;
}

2) Areas nomeadas: layout sem dor

grid-template-areas deixa o layout legivel e facil de manter. Excelente para wireframes e paginas completas.

header
sidebar
main
footer
.layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }

3) Responsividade moderna com auto-fit + minmax

Para grids de cards que se ajustam sozinhos, prefira repeat(auto-fit, minmax(130px, 1fr)).

.cards {

  grid-template-columns: repeat(auto-fit, minmax(130px, 1fr));
  gap: 0.55rem;
}

4) Alinhamento no Grid

justify-content e align-content alinham a grade inteira. Para cada item individual, use justify-self e align-self.

Checklist rapido para lembrar

  • Use Grid para layout 2D; use Flexbox para alinhamentos 1D.
  • Comece por grid-template-columns e gap.
  • Use fr para proporcoes, nao porcentagens fixas.
  • Use auto-fit + minmax para responsividade elegante.
  • Prefira areas nomeadas quando houver cabecalho, menu, conteudo e rodape.
