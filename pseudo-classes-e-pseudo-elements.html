Pseudo-classes e Pseudo-elements em CSS

Este mini site te ensina a diferenca entre pseudo-classes e pseudo-elements com exemplos reais. A regra rapida e: pseudo-class muda o estado de algo que ja existe; pseudo-element cria/estiliza uma parte virtual do elemento.

Pseudo-class: :hover, :focus, :nth-child() Pseudo-element: ::before, ::after, ::first-letter

1) Pseudo-classes (com um : )

Uma pseudo-class seleciona um estado especial de um elemento. Exemplo: quando o mouse passa por cima, quando um campo recebe foco, quando um item e o primeiro/ultimo da lista.

:hover, :active e :focus-visible

.btn-demo:hover { background: #c2410c; }
.btn-demo:active { transform: translateY(0); }
.btn-demo:focus-visible { outline: 3px solid #fdba74; }

:visited e :hover em link

.link-demo:hover { border-color: #0f766e; }
.link-demo:visited { color: #0f172a; }

2) Pseudo-elements (com dois :: )

Um pseudo-element permite estilizar partes especificas do conteudo, ou inserir conteudo visual (decorativo) sem colocar isso no HTML.

::before e ::after

.element-demo::before { content: "Novo"; }
.element-demo::after { content: " <- adicionado com ::after"; }

Curso de CSS avancado

::first-letter e ::first-line

.article-demo::first-letter { font-size: 2.2rem; }
.article-demo::first-line { color: #0f766e; }

pseudo-elements sao muito uteis para acabamento visual em textos longos, como capitular em artigos, revistas, blogs e landing pages.

::selection

.selection-demo::selection {
  background: #fcd34d;
  color: #111827;
}

Selecione este texto com o mouse para ver o efeito de ::selection.

3) Diferenca rapida

Tipo O que faz Exemplos
Pseudo-class Seleciona estado/condicao de um elemento que ja existe. :hover, :focus, :visited, :nth-child(2)
Pseudo-element Seleciona uma parte virtual do elemento ou injeta conteudo decorativo. ::before, ::after, ::first-letter, ::selection

4) Regra de ouro para memorizar

Se voce quer reagir a um estado, use pseudo-class. Se voce quer mexer em uma parte interna/virtual do elemento, use pseudo-element.
