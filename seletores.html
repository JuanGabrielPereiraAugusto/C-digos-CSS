Seletores CSS

No CSS possuímos diversos seletores para estilizar elementos HTML

Seletores de tipo

  • seletor por tipo: seleciona elementos com base no seu tipo (ex: p, h1, div)
  • seletor de classe: seleciona elementos com base na classe atribuída a eles (ex: .classe)
  • seletor de ID: seleciona um elemento específico com base no seu ID (ex: #id)
  • seletor de atributo: seleciona elementos com base em um atributo específico (ex: [type="text"])
  • seletor de pseudo-classe: seleciona elementos com base em seu estado ou posição (ex: :hover, :first-child)
  • seletor de pseudo-elemento: seleciona uma parte específica de um elemento (ex: ::before, ::after)
  • seletor universal: seleciona todos os elementos (ex: *)
  • seletor de descendente: seleciona elementos que são descendentes de um elemento específico (ex: div p)
  • seletor de filho: seleciona elementos que são filhos diretos de um elemento específico (ex: div > p)
  • seletor de adjacente: seleciona um elemento que é imediatamente adjacente a outro elemento (ex: h1 + p)
  • seletor de irmão geral: seleciona todos os elementos que são irmãos de um elemento específico (ex: h1 ~ p)
  • seletor de grupo: seleciona vários elementos de uma vez (ex: h1, h2, p)

Exemplos de Combinadores

Combinador Descendente (espaço)

Seleciona TODOS os elementos de um tipo dentro de um elemento pai, em qualquer nível de profundidade.

Parágrafo filho direto - será estilizado

Parágrafo neto - também será estilizado (é descendente)

Parágrafo bisneto - também será estilizado

Combinador Filho (>)

Seleciona APENAS os elementos diretos (filhos imediatos) de um elemento.

Parágrafo filho direto - será estilizado

Parágrafo neto - NÃO será estilizado (não é filho direto)

Combinador Adjacente (+)

Seleciona um elemento que vem IMEDIATAMENTE depois de outro (irmão adjacente).

Título

Primeiro parágrafo após h4 - será estilizado

Segundo parágrafo após h4 - NÃO será estilizado

Combinador Irmão Geral (~)

Seleciona TODOS os elementos que vêm depois de um elemento específico no mesmo nível.

Outro Título

Primeiro parágrafo após h4 - será estilizado

Algum conteúdo

Segundo parágrafo após h4 - será estilizado

Terceiro parágrafo após h4 - será estilizado
